# Excel Workflow

> **Column layout is project-specific.** The table below reflects the 7631 Creditview workbook.
> For a new project, verify column headers before scripting — don't assume positions.

## Reading the File
Use `win32com` — openpyxl `data_only=True` returns None for array formulas:
```python
import win32com.client
xl = win32com.client.Dispatch('Excel.Application')
wb = xl.Workbooks.Open(r'absolute\path\to\file.xlsx')
ws = wb.Sheets('Sheet1')
value = ws.Cells(row, col).Value
wb.Close(False)
```

## Key Formulas
- K (TPZ Circle Radius m): clean TPZ radius — use as `radius_m * 4.0` to get diameter in mm at 1:500
- R (Trunk): `= MAX(M / 50, floor)` — use col M (multistem calc), not raw DBH
- Convert mm → Illustrator pts: `mm × (72 / 25.4)`

## Notes
- Remove trees: TPZ = "N/A", col J errors out — use col K for size; col K should still have a value
- Multistem DBH string in col D: use col M for calculations, it normalises to a single equivalent value

## data.csv Schema
Generated by `export_data.py`; consumed by `place_tpz.py`, `place_labels.py`, `review.py`.

```
tree_num, direction, tpz_m, tpz_circle_m, cx, cy, trunk_mm
```

| Column | Source | Notes |
|---|---|---|
| tree_num | col A | String |
| direction | col I | "Protect" / "Injury" / "Remove" / "Retain" |
| tpz_m | col J | Display value — may be empty for Remove trees |
| tpz_circle_m | col K | Clean radius in m — multiply by 4.0 for diameter mm |
| cx | col P | Illustrator artboard X (after transform if needed) |
| cy | col Q | Illustrator artboard Y (after transform if needed) |
| trunk_mm | col R | Trunk circle diameter in mm |

Run `python export_data.py` after any Excel edit to regenerate. `place_tpz.py` warns if xlsx is newer than csv.
